# 🎬 Film Production Dashboard

An interactive Streamlit dashboard for analyzing film production data, including budget, revenue, and genre distribution.

## 📊 Overview

This project visualizes basic film industry data and allows users to explore financial performance through interactive charts. Users can also add new films dynamically using the sidebar.

## ✨ Features

* 📁 View film dataset in a table
* ➕ Add new film data (title, genre, budget, revenue)
* 📊 Compare **Budget vs Revenue** with bar charts
* 🥧 Explore **Genre Distribution** with pie charts
* 📈 Automatically calculate **Profit** and **ROI**

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas
* Plotly

## 🚀 How to Run

1. Install required libraries:

```
pip install -r requirements.txt
```

2. Run the app:

```
streamlit run app.py
```

## 📂 Project Structure

* `app.py` : main application file
